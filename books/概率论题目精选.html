第一章 古典概率

难度：基础（高中 / 竞赛）

1.1 等可能事件与古典概型

问题 1 掷一枚均匀骰子，求点数为偶数的概率。

【解答】

样本空间 Ω = {1,2,3,4,5,6}，共 6 个等可能基本事件。 偶数事件 A = {2,4,6}，|A| = 3。 故 P(A) = 3/6 = 1/2。

问题 2 从 1 到 10 中随机取一个整数，求该数能被 3 整除的概率。

【解答】

能被 3 整除的数：{3, 6, 9}，共 3 个。 P = 3/10 = 0.3。

问题 3 袋中有 5 个红球和 3 个白球，随机取一个，求取到红球的概率。

【解答】

共 8 个球，取到红球的概率 P = 5/8。

1.2 排列组合与计数

问题 4 将 4 本不同的书随机排成一排，求某两本特定书相邻的概率。

【解答】

总排列数：4! = 24。 将两本书捆绑视为一个对象：3! 种排法，两书内部 2! 种，共 3! × 2! = 12 种。 P = 12/24 = 1/2。

问题 5 从 52 张标准扑克牌中随机抽 5 张，求 5 张全为同花（同一花色）的概率。

【解答】

总选法：C(52,5) = 2598960。 同花选法：4 种花色，每种 C(13,5) = 1287 种，共 4 × 1287 = 5148 种。 P = 5148/2598960 ≈ 0.00198。

第二章 条件概率与贝叶斯定理

难度：基础至中级（高中 / 本科）

2.1 条件概率

问题 6 掷两枚骰子，已知两数之和大于 7，求两数之和等于 10 的概率。

【解答】

和 > 7 的情形（共 15 种）：(2,6),(3,5),(3,6),(4,4),(4,5),(4,6), (5,3),(5,4),(5,5),(5,6),(6,2),(6,3),(6,4),(6,5),(6,6)。 和 = 10 的情形：(4,6),(5,5),(6,4)，共 3 种。 P(和=10 | 和>7) = 3/15 = 1/5。

问题 7 一家庭有两个孩子，已知至少有一个是女孩，求两个都是女孩的概率。

【解答】

等可能样本空间：{男男, 男女, 女男, 女女}。 至少一女：{男女, 女男, 女女}，共 3 种。 两女：{女女}，1 种。 P = 1/3。

2.2 全概率公式与贝叶斯定理

问题 8 某工厂有甲、乙两条流水线，甲线产品占 60%，乙线占 40%。甲线次品率

2%，乙线次品率 5%。随机抽一件产品，求是次品的概率。

【解答】

设 A = 次品，B1 = 甲线，B2 = 乙线。 P(A) = P(A|B1)P(B1) + P(A|B2)P(B2) = 0.02 × 0.6 + 0.05 × 0.4 = 0.012 + 0.020 = 0.032。

问题 9 接上题，若抽到一件次品，求它来自甲线的概率。

【解答】

由贝叶斯定理： P(B1|A) = P(A|B1)P(B1) / P(A) = 0.012 / 0.032 = 3/8 = 0.375。

问题 10 某病发病率 0.1%，检测灵敏度 99%（患者检测阳性概率），特异度

95%（健康者检测阴性概率）。某人检测阳性，求其真正患病的概率。

【解答】

设 D = 患病，T+ = 阳性。 P(D) = 0.001，P(T+|D) = 0.99，P(T+|D^c) = 0.05。 P(T+) = 0.99×0.001 + 0.05×0.999 = 0.00099 + 0.04995 = 0.05094。 P(D|T+) = 0.00099 / 0.05094 ≈ 1.94%。 （即使检测阳性，患病概率仍不足 2%——基率谬误的经典示例。）

第三章 随机变量与概率分布

难度：中级（大学本科）

3.1 离散随机变量

问题 11 投掷公平硬币 4 次，设 X 为正面朝上次数，写出 X

的概率分布表，并求期望与方差。

【解答】

X ~ B(4, 1/2)。 P(X=k) = C(4,k) × (1/2)^4，k = 0,1,2,3,4。 P(0)=1/16, P(1)=4/16, P(2)=6/16, P(3)=4/16, P(4)=1/16。 E(X) = np = 4 × 1/2 = 2。 Var(X) = np(1-p) = 4 × 1/2 × 1/2 = 1。

问题 12 某呼叫中心每分钟收到的电话数服从泊松分布，平均每分钟 3

个。求某分钟恰好收到 5 个电话的概率。

【解答】

X ~ Poisson(λ=3)。 P(X=5) = e^(-3) × 3^5 / 5! = e^(-3) × 243 / 120 ≈ 0.04979 × 2.025 ≈ 0.1008。

问题 13 独立重复试验中，每次成功概率 p=0.3。求首次成功恰好在第 4 次的概率。

【解答】

X 服从几何分布：P(X=k) = (1-p)^(k-1) × p。 P(X=4) = 0.7^3 × 0.3 = 0.343 × 0.3 = 0.1029。

3.2 连续随机变量

问题 14 设 X ~ U(0, 10)（均匀分布），求 P(3 < X < 7)。

【解答】

f(x) = 1/10，0 < x < 10。 P(3 < X < 7) = ∫[3 to 7] (1/10) dx = 4/10 = 0.4。

问题 15 设 X ~ N(0,1)（标准正态），求 P(-1.96 < X < 1.96)。

【解答】

查标准正态表：Φ(1.96) ≈ 0.9750。 P(-1.96 < X < 1.96) = 2Φ(1.96) - 1 = 2×0.9750 - 1 = 0.95。 （这是正态分布 95% 置信区间的基础。）

问题 16 设 X ~ Exp(λ=2)（指数分布），求 P(X > 1) 及 E(X)。

【解答】

f(x) = 2e^(-2x)，x > 0。 P(X > 1) = ∫[1,∞] 2e^(-2x) dx = e^(-2) ≈ 0.1353。 E(X) = 1/λ = 1/2 = 0.5。

第四章 多维随机变量

难度：中高级（本科 / 研究生）

4.1 联合分布与边缘分布

问题 17 设 (X,Y) 的联合概率密度为 f(x,y) = 6xy，0 < x < 1，0 < y < 1，x+y < 1，其余为

0。求边缘密度 f_X(x)。

【解答】

f_X(x) = ∫[0 to 1-x] 6xy dy = 6x × [y^2/2]_0^(1-x) = 6x × (1-x)^2/2 = 3x(1-x)^2，0 < x < 1。

问题 18 设 X 与 Y 相互独立，X ~ N(0,1)，Y ~ N(0,1)，求 P(X^2 + Y^2 < 4)。

【解答】

X^2 + Y^2 ~ χ^2(2)，等价于参数 λ=1/2 的指数分布。 设 R^2 = X^2 + Y^2，则 P(R^2 < 4) = 1 - e^(-4/2) = 1 - e^(-2) ≈ 0.8647。

4.2 协方差与相关系数

问题 19 设 X ~ U(-1,1)，Y = X^2。证明 X 与 Y 不相关，但不独立。

【解答】

Cov(X,Y) = E(XY) - E(X)E(Y)。 E(X) = 0（对称），E(X^3) = 0（奇函数在对称区间积分为 0）。 故 Cov(X,Y) = E(X·X^2) - 0 = E(X^3) = 0，即不相关。 但 Y 完全由 X 决定（Y = X^2），故不独立。 （说明：不相关不等于独立，仅对正态分布二者等价。）

第五章 大数定律与中心极限定理

难度：高级（本科 / 研究生）

5.1 大数定律

问题 20 某保险公司承保 10000 位客户，每位客户年赔付额 X_i 满足 E(X_i)=500

元，Var(X_i)=40000 元^2，且相互独立。求年总赔付额超过 510 万元的概率近似值。

【解答】

设 S = ΣX_i，n=10000。 E(S) = 10000 × 500 = 5,000,000。 Var(S) = 10000 × 40000 = 4×10^8，标准差 σ_S = 20000。 由 CLT：Z = (S - 5000000)/20000 ~ N(0,1) 近似。 P(S > 5100000) = P(Z > (5100000-5000000)/20000) = P(Z > 5) ≈ 0.0000003。 （超赔风险极小，体现保险大数法则。）

5.2 中心极限定理应用

问题 21 某网站每次页面加载时间服从均值 2 秒、标准差 0.5

秒的分布（不一定正态）。随机观测 100 次，求样本均值超过 2.1 秒的概率。

【解答】

由 CLT，样本均值 X_bar ~ N(2, (0.5)^2/100) = N(2, 0.0025)，标准差 0.05。 P(X_bar > 2.1) = P(Z > (2.1-2)/0.05) = P(Z > 2) = 1 - Φ(2) ≈ 1 - 0.9772 = 0.0228。

问题 22 用正态近似计算：投硬币 400 次，正面朝上次数在 190 到 210 次之间的概率。

【解答】

X ~ B(400, 0.5)，近似 N(200, 100)，标准差 10。 加连续性修正：P(189.5 < X < 210.5) = P((189.5-200)/10 < Z < (210.5-200)/10) = P(-1.05 < Z < 1.05) = 2Φ(1.05) - 1 ≈ 2×0.8531 - 1 = 0.7062。

第六章 数理统计基础

难度：高级（研究生 / 考研）

6.1 参数估计

问题 23 设总体 X ~ N(μ, σ^2)，取样本 {3.2, 2.8, 3.5, 3.1, 2.9}，求 μ

的极大似然估计量与 95% 置信区间（σ^2 已知 = 0.04）。

【解答】

MLE：μ_hat = X_bar = (3.2+2.8+3.5+3.1+2.9)/5 = 15.5/5 = 3.1。 95% CI（σ 已知）：X_bar ± 1.96 × σ/√n = 3.1 ± 1.96 × 0.2/√5 = 3.1 ± 1.96 × 0.0894 = 3.1 ± 0.175 = (2.925, 3.275)。

问题 24 设 X_1,...,X_n i.i.d. ~ Exp(λ)，求 λ 的矩估计量和 MLE，并比较。

【解答】

矩估计：E(X)=1/λ，令样本均值 X_bar = 1/λ，得 λ_MOM = 1/X_bar。 MLE：对数似然 l(λ) = n ln λ - λ Σx_i， 令 dl/dλ = n/λ - Σx_i = 0，得 λ_MLE = n/Σx_i = 1/X_bar。 结论：两种方法得到相同估计量，均为 1/X_bar。

6.2 假设检验

问题 25 某药物声称平均降压幅度 10 mmHg，某研究者对 25 名患者试验，得样本均值

9.2 mmHg，样本标准差 2 mmHg。在显著性水平 α=0.05

下，检验该药是否达标（双侧检验）。

【解答】

H0: μ=10，H1: μ≠10。σ 未知，用 t 检验。 t = (X_bar - μ0) / (s/√n) = (9.2 - 10) / (2/√25) = -0.8/0.4 = -2.0。 自由度 df = 24，t_(0.025,24) ≈ 2.064。 |t| = 2.0 < 2.064，不拒绝 H0。 结论：在 5% 显著性水平下，无充分证据认为该药未达标。
